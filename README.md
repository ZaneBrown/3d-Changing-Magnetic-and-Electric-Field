# 3d-Changing-Magnetic-and-Electric-Field
Basic 3d plotting and animation creator for a charged particle in the presence of both an magnetic and electric field.

## Instructions
### Get necessary libraries and constants
Run the first cell to get the necessary libraries and constants

### Run either the 3d Plot Graph or 3d Plot Animation to get a test plot or animation
After testing it with verified functions, you can change many values, such as initial position, velocity, acceleration, time steps, initial B and E fields and the functions that vary B and E fields over position.
Most other functions may diverge it given enough time which will make the plot or animation boring. The test cases uses sine and cosine to keep the particle close to the starting position, but experimenting with different math functions can be fun to see how to balance the fields.
